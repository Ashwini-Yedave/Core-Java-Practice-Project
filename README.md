# Student Management System - Core Java

Console-based Java application for managing student records.

## Features
- Add Student
- Show All Students
- Find Topper
- Calculate Average Marks
- Search Student by ID
- Delete Student
- Exception Handling

## Technologies Used
- Java 17
- OOP Concepts
- ArrayList
- Scanner Class

## Project Structure

Student.java
Main.java

## How to Run

Compile:
javac Student.java Main.java

Run:
java Main

- Classes & Objects
